3. Teori : 
- apa itu git ?
Git adalah sebuah tool yang sering digunakan dalam proyek pengembangan software dan banyak digunakan oleh programer.
- untuk apa fungsi dari command :
    + git init 
        digunakan untuk membuat repositori lokal yang nantinya terdapat folder .git
    + git remote add origin <link_name>
        untuk menambahkan file/data ke Github/Gitlab 
    + git add <file_name>
        untuk menambahkan file/data pada repo yang dipilih
    + git commit -m "<message_name>"
        untuk menyimpan perubahan pada file/data
    + git push origin master
        untuk menambahkan file/data ke lokal server

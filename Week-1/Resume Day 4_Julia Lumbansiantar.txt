Resume Day 4
Basic JavaScript(JS)

Variable = wadah data dimuat
Cara penulisan Variable
1. Camel case : var addProduct = []
2. Snake case : var add_product = []
3. Kebab case : var add-product = []
4. Pascal case : var AddProduct = []

Tipe data
1. String : kata atau rangkaian kata atau kalimat
2. Number : angka 0-9
3. Boolean : True or False
4. Null
5. Undefined : tidak terdefenisikan karena variable tidak dideklarasi

Array = kumpulan beberapa variable.

Tipe angka
1. Desimal 0-9
2. Biner (mirip seperti Boolean karena memiliki dua nilai saja) 0 1
3. Octal 0-7
4. Hexadecimal 0-9 ABCDEF

Operasi Matematika
1. Tambah
2. Kali 
3. Bagi 
4. Kurang
5. Modulus (sisa pembagian. Biasa dilambangkan dengan %)
